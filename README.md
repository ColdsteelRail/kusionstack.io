# KusionStack 文档站点

该仓库用于构建 https://kusionstack.io 文档网站，站点基于 [Docusaurus 2](https://docusaurus.io/) 构建。

## 文档目录

- `/docs` - 文档根目录
- `/docs/user_docs` - 使用文档
- `/docs/develop` - 开发文档
- `/docs/referece` - 参考手册
- `/docs/governance` - 治理
- `/blog` - 博客


## 本地预览

可以通过以下命令在本地启动文档服务：

```
$ npm install
$ npm run start
```

默认会打开 http://localhost:3000 文档页面。
